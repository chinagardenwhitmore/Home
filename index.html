my work.

Bias Bot

My friends and I were curious to see if we could somehow quantify a prevalent problem today, bias in news. Using machine learning, my team and I were able to (mostly) do exactly that. After Bias Bot predicts how an article spins the topic, it provides the user with a few articles to round their viewpoint out. While Bias Bot is still very much a work in progress, it gave the team a great introduction to building something cool with machine learning.

Pomodoro Timer

There's a pretty good chance you've heard of the Pomodoro technique, and that's because it works. The problem I faced when using the Pomodoro technique was finding background music and setting a timer took too much time away from productivity. The Pomodoro Timer asks the user how long they want to be productive for, break duration, and a music genre. From there, it does the heavy lifting and lets the user focus on productivity.

2048

2048 is the only project from my coursework included on my site, but I feel as if I had to because I may have had a bit too much fun with it. The original 2048 consumed my entire high school when it was hot, so I felt that if I was going to clone it, I had to do it justice. On top of the base requirements, I included a leaderboard, color changing tiles and sliding tiles. This is written in Java and I opted to use JavaFX instead of Swing for the front end.
